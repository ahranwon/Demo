Claire Fant, Instructor

My name is Claire Fant, and I am the instructor for VCT 170 - Interactive / Web Design. I like the integration of design and technology that is the web. As visual designers we can enhance communication, make it useful, fun and purposeful. The technology of the web has become more complex, but the basics are the same. Complexity arises from building upon the basics. So we will focus on the basics.

HTML, CSS & JavaScript

HTML stands for hypertext markup language. HTML is what makes the web what it is. It is what allows us to jump from one document to another, regardless of their location in the world. HTML gives a meaningful structure to a site’s content so that it is accessible and understandable. Applying a meaningful structure means that the content can be ported to different platforms and retain its meaning.

CSS stands for Cascading Style Sheets. It controls the appearance of the basic HTML structure so that communication is enhanced.

JavaScript takes care of the complex behavior of HTML and CSS by incorporating dynamic actions and animations based upon user actions.

Late 1990s

When I first learned how to lay out and build websites, we graphic designers would use mostly images and HTML tables to position content. At the time they worked well for simple layouts. However, as web use evolved and grew more complex, using tables for positioning content became unwieldy and time-consuming.

2000s

As hardware technology and web use continued to progress by leaps and bounds, it made sense to the web design community that utilizing web standards was the best way to accommodate future changes. The cornerstone of web standards is the separation of content and presentation. CSS was developed to be able to handle more sophisticated presentations of HTML content.

Web 2.0

The main purpose of the web had always been to link documents over the internet. Over the years it has evolved from being a static purveyor of information to a platform where interactivity takes on many more forms, such as video streaming and social sharing.

The web is also accessed via a variety of devices such as smart phones and tablets. As a result the W3C has published recommendations for HTML5 and CSS3, which allow designers/developers to format content for presentation across multiple devices and to handle more types of content and implement greater interactivity. CSS3 is gradually taking on some tasks that used to be only accomplished with JavaScript, such as simple animations.

The scripting language, JavaScript (official name is ECMA Script) has a greater stake in building web applications (software in the browser). In this course we will mostly focus on the fundamental concepts of building a static website, but will touch on a little JavaScript.
